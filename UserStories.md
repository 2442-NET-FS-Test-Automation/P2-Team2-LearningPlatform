# LearnHub User Stories

## Admins

- **As an Admin**, I want to log in and out securely so that I can access the administrative dashboard safely.
- **As an Admin**, I want to create, update, and delete user accounts so that I can control system access and assign appropriate roles.
- **As an Admin**, I want to create, update, and delete courses so that I can maintain an up-to-date and accurate course catalog.
- **As an Admin**, I want to create, update, and delete professor profiles so that I can manage the teaching staff effectively.
- **As an Admin**, I want to create, update, and delete shifts so that I can organize and maintain the class schedule structure.

---

## Managers

- **As a Manager**, I want to log in and out securely so that I can access administrative tools safely.
- **As a Manager**, I want to create, update, and delete courses so that I can keep the course catalog current and relevant.
- **As a Manager**, I want to create, update, and delete professor profiles so that I can assign and manage faculty resources.
- **As a Manager**, I want to create, update, and delete shifts so that I can define and adjust class schedules as needed.

---

## Professors

- **As a Professor**, I want to log in and out securely so that I can access my personal teaching dashboard.
- **As a Professor**, I want to edit my own profile information so that I can keep my contact details and professional bio up-to-date.
- **As a Professor**, I want to view my assigned courses so that I can clearly see my teaching schedule and responsibilities.
- **As a Professor**, I want to set and update grades for students so that I can accurately record their performance and provide feedback.
- **As a Professor**, I want to view each student's enrollment and progress status so that I can monitor participation and identify students who may need extra support.

---

## Students

- **As a Student**, I want to log in and out securely so that I can access my personal learning portal.
- **As a Student**, I want to edit my own profile information so that I can keep my contact details and preferences current.
- **As a Student**, I want to view the full course catalog so that I can explore all available learning opportunities.
- **As a Student**, I want to see detailed information about each course (e.g., syllabus, prerequisites, schedule) so that I can make well-informed enrollment decisions.
- **As a Student**, I want to enroll in available courses so that I can register for the classes I wish to take.
- **As a Student**, I want to unenroll from courses I am currently in so that I can drop classes I no longer wish to pursue.
- **As a Student**, I want to see my grades for active courses so that I can track my academic performance throughout the term.
- **As a Student**, I want to see my completion progress for enrolled courses so that I can monitor how much material I have covered and what remains.

---

## Anonymous Users

- **As an Anonymous User**, I want to register for an account so that I can gain full access to the learning platform and enroll in courses.
- **As an Anonymous User**, I want to browse the course catalog with limited preview information so that I can explore what the platform offers before committing to registration.
